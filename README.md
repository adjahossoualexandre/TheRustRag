# Desccription

A Retrieval-Augmented Generation (RAG) chatbot based on [the Rust book](https://github.com/rust-lang/book)

# Instruction

execution order:

    `pip install -r requirements.txt`
    `python data_processing.py`
    `python indexing.py`
    `python querying.py`
